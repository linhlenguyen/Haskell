Translating word and power point to html

This activity is to give you practice translating word and power point documents to html. You may want to do this when you create your portfolio.

Refer to the notes while you do this activity.

  • Create a new directory on your H drive.
  • Find a word document and a power point document to practice with. Copy them to this new directory. You don't want to risk changing them, so make a copy.
  • Following the steps in the notes translate both to html files.
  • Create a small html file. You will place links in this file to the files you created in the translation process.
    You may use this link to get a copy of a simple file.
    1. Right click on the link
    2. choose "Save Target As ..."
    3. Save it in the newly created directory
    4. Edit it with Notepad or some other editor.
  • Put links in the file, and then double click it. Click on the links you created and see how your translated files appear.
