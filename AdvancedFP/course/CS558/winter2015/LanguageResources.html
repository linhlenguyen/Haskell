    CS 558 Language Resources
    • Python
      • Python3 is available on the CS linuxlab machines as python3
      • Official Python website
      • The current release (3.4.2) can be downloaded from here
      • Documentation index for version 3.4.2.
        • The tutorial is strongly recommended for beginnners.
        • You will also need to refer frequently to the standard library reference.
        • Less frequently, you'll need to consult the language reference manual.
      • There are lots of tutorial and reference books on Python. Some suggestions will appear here soon...

    • Haskell
      • Haskell is available on the linuxlab machines as ghci
      • The official Haskell web site.
      • The Haskell platform, the best way to download and install Haskell.
      • Some tutorials on using Haskell
      • Tim Sheard's Haskell resource page.

    • OCaml
      • We won't be using O'Caml this quarter, winter 2015, but this is here for completeness.
      • OCaml version 4.00 is now available on the CS linuxlab machines. To enable it, use the addpkg command (more information). If you don't load this package, you'll get version 3.12.1 instead...
      • Unofficial OCaml website. (There's a lot of useful information here, but it is less authoritative than the inria.fr sites referenced below.)
      • Current release (4.00.1) can be downloaded from here.
      • Documentation for release 4.00.
        • The introduction to the core language is strongly recommended.
        • You will want to become familiar with the language reference manual...
        • ...and with the core library and the standard library .
        • There are not many published books on OCaml, but you may find this on-line book by Jason Hickey useful.
