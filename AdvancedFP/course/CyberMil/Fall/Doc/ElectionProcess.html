What do we want out of a voting machine

  • Easy to use
  • Accurate
  • Easy to set up
  • Creates an audit trail

The Election Process in the US for presidential elections

  • Create an election bureaucracy
  • Set up a voter rolls
  • process for adding eligible voters
  • process for removing ineligible voters
  • create a ballot
  • certify voter as eligible
  • distribute ballot
  • voter marks ballot
  • ballots are collected
  • votes are counted (discern voter intent, counted by hand or machine)
  • votes are accumulated at the local level
  • votes are accumulated at the state level
  • electors are certified at the state level
  • electors vote, and counts are accumulated
  • winner is declared
