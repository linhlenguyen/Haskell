Group Activity Evaluation Questions

How did the groups go

  1. What was good?
  2. What was bad? How could we improve it.
  3. Where did we waste time?
  4. Did everyone contribute?
  5. Were some people unprepared (I.e. they didn't do the reading)

Some thoughts on the group writing process.

  1. How did you write your 100-250 words?
  2. Did you write several drafts?
  3. Did you make an outline?
  4. Did you word-smith it?
  5. what else of note happened?
