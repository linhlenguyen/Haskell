Turning in Programming Assignments

When you turn in a programming assignment there are three things I need.

  • A copy of the program on paper, so that I can mark it up, and return to you.
  • A copy of the output, so that I can see that it works correctly
  • An actual copy of the program so I can run it.

Here is how I want you to proceed.

  • The paper copy. Print it out and bring it to class.
  • The output.
    1. Run the program in a freshly opened winhugs.
    2. Run your program on enough inputs to demonstrate that it works.
    3. Use the clip board to copy the complete buffer
    4. Paste the buffer into a blank Notepad editor file.
    5. Print the file and hand the copy of the output to me in class along with the copy of the program.
    6. Write your name on both parts and staple them together
  • The actual program.
    1. Email me the program as an attachment
    2. The body of the email should include who the programmers are.
    3. Be sure your name(s) are in the program as comments as well.
    4. The name of the attachment should include the name of the assignment as well as the programmers name
    5. For example "calendarTomRyan.hs"
